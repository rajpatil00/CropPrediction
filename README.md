# 🌾 Crop Prediction System

A Machine Learning-based web application that predicts the most suitable crop based on environmental conditions like temperature, humidity, rainfall, and soil type. The project uses a trained classification model and is deployed using Flask.

---

## 🚀 Features
- Predicts suitable crop based on input values
- Machine Learning classification model
- Flask web application
- Simple and user-friendly interface
- Fast prediction results

---

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML, CSS
